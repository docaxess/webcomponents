<ip-tooltip
            id="clicked-tooltip" tooltip-trigger="Click me !"
            tooltip-content="Please enter a description of the tooltip. The length
            is 3/4 lines maximum. Arrows can be adjusted position."
            tooltip-btn-close="true" 
            tooltip-btn-1="Cancel" 
            tooltip-btn-2="Learn More" 
            type="click" 
            > 
            </ip-tooltip>

Installation

Follow these steps to install and use the ip-tooltip component in your project.

Step 1

Install the ip-tooltip component as a dependency in your project: 

npm install ip-tooltip

Step 2

Import the module into your script file:

import '../node_modules/ip-tooltip/dist/ip-tooltip/ip-tooltip.esm';

Usage

To use these components, you must add the properties as attributes.

Property Attribute Required Description Type Default Value
tooltipTrigger tooltip-trigger Yes The button that will display the tooltip string undefined
tooltipContent tooltip-content Yes The text that will be displayed in the tooltip string undefined
type type No Determines if the tooltip should appear on click or hover "click" \| "hover" "hover"
tooltipBtnClose tooltip-btn-close No If you need to add a close button in the tooltip boolean false
tooltipTitle tooltip-title No The title of the tooltip if you want to add a title string undefined
tooltipBtn1 tooltip-btn-1 No If you need to add a button in the tooltip string undefined
tooltipBtn2 tooltip-btn-2 No If you need to add a second button in the tooltip string undefined
btn1AriaLabel btn-1-aria-label No ARIA label for the first button, required for accessibility string undefined
btn2AriaLabel btn-2-aria-label No ARIA label for the second button, required for accessibility string undefined

Customization

We have a set of predefined variables used to customize the tooltip:

  • --primary-color
  • --secondary-color

To update the values use the following:


                ip-tooltip {
                  --primary-color: #006342;
                  --secondary-color: #000000;
                  }

Listening to Events

To listen to events emitted by ip-tooltip, open your browser's console. Event details will be displayed in the console when the buttons are interacted with. 


            document.addEventListener("DOMContentLoaded", function () {
            const tooltip = document.querySelector("#clicked-tooltip");
            tooltip.addEventListener("btn1Click", function () {
            console.log("Event btn1Click captured from index.html");
            });
            tooltip.addEventListener("btn2Click", function () {
            console.log("Event btn2Click captured from index.html");
            });
            document.querySelector("#btn1").addEventListener("click", function () {
            const event = new CustomEvent("btn1Click");
              tooltip.dispatchEvent(event);
            });
            document.querySelector("#btn2").addEventListener("click", function () {
            const event = new CustomEvent("btn2Click");
            tooltip.dispatchEvent(event);
            });
            });
