Installation

Follow these steps to install and use the ip-modal component in your project.

Step 1

Install the ip-modal component as a dependency in your project: 

npm install ip-modal

Step 2

Import the module into your script file:

import '../node_modules/ip-modal/dist/ip-modal/ip-modal.esm';

Usage

To use the ip-show-more component, use this tag <ip-show-more></ip-show-more> and use a slot for the content.

Properties

Property Attribute Necessity Description Type Default
showLessText show-less-text Optional The text for "Show Less" string 'Show Less'
showMoreText show-more-text Optional The text for "Show More" string 'Show More'

Slot

The component uses a slot named content for the expanded content.

e.g:

        <ip-show-more show-more-text="Read More" show-less-text="Read Less">
          <div slot="content">
            This is the expanded content that will be shown or hidden when the button is clicked.
          </div>
        </ip-show-more>

Customization

We have a set of predefined variables used to customize the show-more button:

  • --ip-primary-color
  • --ip-secondary-color
  • --ip-font-size
  • --ip-font-color
  • --ip-font-family
  • --ip-svg-color

e.g:

To update the values, use the following CSS code:

        ip-show-more {
          --ip-primary-color: #006342;
          --ip-secondary-color: #000000;
          --ip-font-size: 16px;
          --ip-font-color: #333;
          --ip-font-family: Arial, sans-serif;
          --ip-svg-color: #fff;
          }
