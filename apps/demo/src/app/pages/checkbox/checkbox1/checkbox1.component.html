Check me !

Installation

Follow these steps to install and use the ip-checkbox component in your project.

Step 1

Install the ip-checkbox component as a dependency in your project: 

npm install ip-checkbox

Step 2

Import the module into your script file:

import '../node_modules/ip-checkbox/dist/ip-checkbox/ip-checkbox.esm';

Usage

To use these components, you must add the properties as attributes.

Properties

Property Attribute Required Description Type Default
id id Yes The ID of the input for accessibility string undefined
name name Yes The name of the input for accessibility string undefined
defaultChecked default-checked No Sets the checkbox to checked by default boolean false
disabled disabled No Disables the checkbox boolean false
Content

There is a predefined label slot for the label before the checkbox, which will be the content inside the tag.

Example:
<ip-checkbox default-checked="true" disabled="true" id="checked"> Checked ! </ip-checkbox>

Customization

We have a set of predefined variables used to customize the checkbox:

  • --primary-color
  • --secondary-color
  • --font-size
  • --checkbox-size

To update the values, use the following CSS:

ip-checkbox {
          --primary-color: #006342;
          --secondary-color: #000000;
        }

Listening to Events

To listen to events emitted by the ip-checkbox and ip-checkbox-list components, open your browser's console. Event details will be displayed in the console when checkboxes are modified.

Example JavaScript Code:

    const checkbox = document.querySelector('ip-checkbox');
    checkbox.addEventListener('change', event => {
      const isChecked = event.detail.checked;
      console.log(isChecked);
    });
